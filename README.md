# Sorteador de Numeros

![Capa do projeto Sorteador de Numeros](./image/capa.png)

Uma aplicacao web interativa para sortear numeros de forma rapida, bonita e intuitiva. O usuario define a quantidade, escolhe o intervalo e pode optar por evitar repeticoes no resultado.

## Visualizar projeto

[Acessar versão online](https://eloaguilgel.github.io/sorteador-de-numeros/)

## Sobre o projeto

O **Sorteador de Numeros** foi desenvolvido com foco em uma experiencia simples e agradavel. Alem do sorteio em si, a interface apresenta animacoes de resultado, feedback de validacao e uma navegacao bem direta entre o formulario e a tela final.

## Funcionalidades

- Sorteio de `1` a `4` numeros por vez
- Definicao livre de intervalo minimo e maximo
- Opcao para nao repetir numeros
- Validacao dos campos antes do sorteio
- Exibicao animada dos resultados
- Botao para sortear novamente com as mesmas configuracoes
- Layout responsivo

## Tecnologias usadas

- `HTML5`
- `CSS3`
- `JavaScript`

## Como executar localmente

1. Clone este repositorio:

```bash
git clone https://github.com/eloaguilgel/sorteador-de-numeros.git
```

2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` no navegador.
